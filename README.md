# Sistema Inteligente de Transporte Masivo

## Integrantes
- Jose Andres de la Ossa Carmona

## Descripción
Sistema basado en reglas lógicas que utiliza búsqueda heurística (A*) para encontrar la mejor ruta.

## Cómo ejecutar
1. Instalar Python 3.x
2. Ejecutar: python sistema_transporte.py

## Base de Conocimiento
El conocimiento está representado en la lista `base_de_conocimiento` usando objetos `ReglaConexion`.
